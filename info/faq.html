GPLv3 is a license that tells others how they can use the source code of my website. (The following is not legal advice)
    GPLv3 allows the following:
  • Freedom to access/use/modify the source code of this website for any purpose.
  • Freedom to redistribute any modified or unmodified source code, as long as the distribution is also under the GLPv3 license.
  • Using DRM or other hardware restrictions to attempt prevent users from modifying the source code.
    GPLv3 does not allow the following:
  • Distributing the source code under a closed source/more restrictive license.
  • Enforcing the legal protections of a standard DRM by way of the DMCA act or other anti-circumvention laws.
  • Restricting access to the source code more than GPLv3 allows.
For more information on the specifics of what GPLv3 does and doesn't allow, check out the GPL website
